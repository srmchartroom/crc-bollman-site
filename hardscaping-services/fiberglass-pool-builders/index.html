The Design-Build Process

What to Know Before Fiberglass Pool Installation

A pool should feel like it belongs to the property, not like it was simply placed where it fit. Placement, circulation, grade, retaining walls, drainage, surrounding patio work, and final hardscape details all have to be considered together so the finished space works as one complete environment.

As experienced fiberglass pool builders and fiberglass pool installers, BOLLMAN approaches each project with both design vision and installation discipline — because a pool that looks beautiful still has to be set correctly, supported properly, and integrated with the rest of the environment.

Elements Often Planned Alongside the Pool

  • Surrounding patios and pool decks
  • Walkways and transitions
  • Retaining walls and grading corrections
  • Drainage improvements
  • Pool coping and matching hardscape finishes
  • Fire features and seating areas
  • Integrated lighting
  • Expanded entertaining space planned by experienced pool and patio contractors

For homeowners exploring custom fiberglass pools, the most important thing to understand is that the pool itself is only part of the project. The finished result depends on how well the design, engineering, build, and finish work all come together.
